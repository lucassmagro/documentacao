## Resumo da matéria

Banco de Dados I introduz o **modelo relacional** e como armazenar, organizar e
consultar dados de forma consistente. Cobre a **modelagem** (modelo
entidade-relacionamento — entidades, atributos, relacionamentos, cardinalidade),
a conversão para o **modelo relacional** e a **normalização** (1FN, 2FN, 3FN para
reduzir redundância), além da linguagem **SQL** (DDL para criar estruturas, DML
para manipular dados, consultas com `SELECT`, `JOIN`, agregações).

É a base prática para os projetos que usam PostgreSQL e para entender integridade,
chaves primárias/estrangeiras e transações.
